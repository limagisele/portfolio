Static Portfolio Website

March 2022

This static website was built with as my 1st assignment of the Coder Academy Bootcamp.
It was built with the intention to practice my HTML and CSS knowledge while applying development tools such as sitemap, wireframes, kanban, responsive design, image optimisation and website deployment.
As an exercise, my main focus was to get comfortable with CSS flexbox in order to achieve a responsive mobile-first design, with different layout for 3 screen sizes.
Now I have started to add a bit of JavaScript as well, just for fun while I'm learning new tools.
